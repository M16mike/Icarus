# ☀️ Solar Yield & Economic Analyzer

An interactive web-based physics simulation engine built with **Python**, **pvlib**, and **Streamlit**. This tool calculates real-world solar energy yields and financial payback performance using historical meteorological datasets.

## 🚀 Features
- **Accurate Solar Physics Engine:** Uses `pvlib` to model atmospheric irradiance conditions, solar positions, and panel cell thermal limits.
- **Real-Time Economic Scaling:** Dynamic calculations for system payback periods, annual utility bill savings, and array output sizing.
- **Interactive Web Interface:** Built with Streamlit to provide quick sliders and immediate visual data charting.

2. Install the necessary Python backend packages:
   ```bash

   ```
3. Boot up the local browser dashboard interface:
